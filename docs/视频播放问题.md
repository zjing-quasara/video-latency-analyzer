# 视频播放问题说明

## 问题
HTML报告中的视频无法在浏览器中播放

## 原因
使用了 MPEG-4 (mp4v) 编码器，部分浏览器不支持

## 解决方案

### 方案1: 直接播放视频文件（推荐）
在报告文件夹中直接双击 `.mp4` 文件，使用系统默认播放器观看

### 方案2: 安装H.264编码器
Windows系统需要安装额外的编码器库：

1. 下载 OpenH264: https://github.com/cisco/openh264/releases
2. 下载 `openh264-1.8.0-win64.dll`
3. 放到以下目录之一：
   - `C:\Windows\System32\`
   - Python安装目录
   - 项目目录

4. 重新运行分析

### 方案3: 使用FFmpeg转换
```bash
# 安装ffmpeg
# 转换视频为浏览器兼容格式
ffmpeg -i 原视频.mp4 -c:v libx264 -preset fast 输出.mp4
```
